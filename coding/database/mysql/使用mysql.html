# 使用Mysql

不特殊说明：所有操作均在命令行模式下。

# 登陆

# 免交互登录
# -u root 指定登录的账户是“root”
# -padmin321 账户密码是“admin321”，注意-p是小写与-P端口号区别，-p与密码之间没有空格
# -h 192.168.163.111 指定mysql服务端所在的主机
# -P 3306 指明mysql服务所在主机的端口是3306，这是默认端口
# 注意，如果你是访问本机mysql服务，指明-h -P(如果你的端口不是默认3306，需要注意)
$ mysql -u root -padmin321 -h 192.168.163.111 -P 3306

# 密码交互登录
# 访问本机服务，如果不输入密码，会弹出需要键入密码的交互提示
$ mysql -u root -p
Enter password:
