# Centos7-minimal基础环境配置

# 网络

# 常用工具
wget 用来从指定的URL下载文件
gcc glibc gcc-c++ make cmake 编译器，编译安装的时候会用到
net-tools Linux内核中配置网络功能的工具
screen 用于命令行终端切换
lrzsz 拖动上传，sz下载
tree 树形目录
dos2unix Windows格式文件转换为Unix、Linux格式的实用命令
lsof 列出当前系统打开文件的工具
tcpdump 抓包工具
bash-completion 自动补全功能增强
ntp 同步网络事件
psmisc 帮助管理/proc目录的程序
setuptool 安装setup命令工具
bind-utils 解决没有nslookup命令
traceroute 数据包路由追踪
openssl openssl-devel web安全通信的基石，没有openssl，可以说我们的信息都是在裸奔

一键安装

yum -y install wget gcc glibc gcc-c++ make cmake net-tools screen lrzsz tree dos2unix lsof tcpdump bash-completion ntp psmisc openssl openssl-devel setuptool bind-utils traceroute

# 工具

更换yum源为国内源，请先安装wget命令

1. 备份源
mkdir /etc/yum.repos.d/bak
mv /etc/yum.repos.d/*.* /etc/yum.repos.d/bak

2. 配置阿里云yum
wget -O /etc/yum.repos.d/CentOS7-Aliyun.repo http://mirrors.aliyun.com/repo/Centos-7.repo
wget -O /etc/yum.repos.d/epel-7-Aliyun.repo http://mirrors.aliyun.com/repo/epel-7.repo
wget -O /etc/yum.repos.d/docker-ce.repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

3. 清除yum缓存并更新
yum makecache
yum -y update

# vim

1. yum -y install vim*

# 添加如下配置
2. vim /etc/vimrc
set nu          " 设置显示行号
set showmode    " 设置在命令行界面最下面显示当前模式等
set ruler       " 在右下角显示光标所在的行数等信息
set autoindent  " 设置每次单击Enter键后，光标移动到下一行时与上一行的起始字符对齐
syntax on       " 即设置语法检测，当编辑C或者Shell脚本时，关键字会用特殊颜色显示
