# docker数据卷管理

# 将Docker主机数据挂载到容器

Docker提供三种不同的方式将数据从宿主机挂载到容器中：volumes，bind mounts和tmpfs。
volumes：Docker管理宿主机文件系统的一部分（/var/lib/docker/volumes）。
bind mounts：可以存储在宿主机系统的任意位置。
tmpfs：挂载存储在宿主机系统的内存中，而不会写入宿主机的文件系统。

# 用卷创建一个容器：

#命令解析
# docker run -> docker创建并运行一个容器
# -d -it 后端交互式运行
# --name -> 命名容器名
# --mount -> 挂载文件
# src=nginx-vol 宿主机目录(/var/lib/docker/volumes/)下的nginx-vol，没有就创建
# dst=/usr/share/nginx/html 上面的数据卷文件挂载到容器内的/usr/share/nginx/html目录
# nginx 镜像名
# docker run -d -it --name=nginx-test --mount src=nginx-vol,dst=/usr/share/nginx/html nginx


# docker run -d -it --name=nginx-test -v nginx-vol:/usr/share/nginx/html nginx
