# 私有镜像仓库Harbor

# Harbor的由来

Habor是由VMWare公司开源的容器镜像仓库。事实上，Habor是在Docker Registry上进行 了相应的企业级扩展，从而获得了更加广泛的应用，这些新的企业级特性包括：管理用户 界面，基于角色的访问控制 ，AD/LDAP集成以及审计日志等。

# 准备工作

我们本次安装harbor以docker-compose的方式部署。
harbor服务器的最低要求2核4G。
准备一台新服务器。

# 为项目指定拥有者/开发者/浏览者

这里需要注意的是，账号本身只有管理员和非管理角色。
而在给项目指定用户的同时，又可以为用户指定该项目的权限角色。
这里我们把david指定为拥有mytest项目开发者的权限。
